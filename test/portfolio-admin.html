Iceball
Mijn eerste project was een minigame-plugin voor Minecraft servers. De plugin heet Iceball, wat afgeleid is van de naam van de server.
De plugin doet veel verschillende taken zoals het automatisch registreren van spelers in een databank, ook heeft het een ingebouwde coinAPI die verbonden kan worden met andere plugins, en een minigameAPI waarmee je minigames kan bouwen om vervolgens mee te kunnen spelen voor je vrienden. Het heeft ongeveer anderhalf jaar geduurd om de plugin de bouwen.
Het project heeft 10.000 regels aan code en dat maakt het mijn grootste project.
